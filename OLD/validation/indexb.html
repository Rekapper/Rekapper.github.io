Assignment Instructions

While it may sound simple to validate your HTML code, it is often times difficult to understand the error messages validation services return. Using the W3C Markup Validation Service, validate and correct the errors in each of these five web pages. After saving the files to your computer, review each of them and correct as many errors as you can BEFORE validating. Then, upload each file to the validation service. You will need to locate and correct the errors in each file. Once you have finished:

  • upload each of the corrected files to Canvas via the Upload File tab
  • add multiple files by clicking the Add Another File button
NOTE: Just because the page may display correctly, does not mean that it is coded correctly.
