Validation in Firefox

If you download the Web Developer Add-on for Mozilla Firefox, you can choose to validate your code without going directly to W3C Validator.

To Validate a Published Web Page:

  1. Open your web page in Firefox
  2. right-click on the page
  3. choose Web Developer from the list, then Tools
  4. Select Validate HTML

To Validate a Local HTML File:

  1. Open your web page in Firefox
  2. on the keyboard, press and hold ALT - SHIFT - L
Both options will give you a validation report using the W3C Validator. You can view the error messages by scrolling down the page.
